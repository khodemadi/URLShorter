# url shorter

کوتاه کننده لینک با PHP ، مای اس کیو ال و Javascript – ساخت یک انتقال دهنده لینک نیازمند این است که راه حلی را داشته باشیم که بتوانیم یک آدرس طولانی را به یک آدرس کوتاه تبدیل کنیم .
فایل index.php
خط 18 : از اونجایی که فرم ajax خواهد بود زمانی که آدرس وارد شد و کاربر ENTER زد نمی خواهیم هدایت بشیم بنابراین با مقدار false برمی گردانیم کــــه ریدایرکت نشیم .
خط 19 : pattern را برای ورودی http و https تنظیم می کنیم که با یکی از این 2 شروع شود .

فایل contants.php برای اطلاعات ثابت مثل دیتابیس

فایل functions.php برای تابع های دیتابیس

فایل func.actions.php
این فایل شامل 3 تابع است که 3 عمل اصلی را هم انجام می دهد :

فانشکن func_action_submit : برای زمانی که لینکی را ثبت می کنیم .
فانشکن func_action_redirect : کاربر را با لینک کوتاه به لینک اصلی هدایت می کند .
فانشکن func_action_redirect_info : اطلاعاتی از لینک کوتاه ثبت شده به ما می دهد شامل تاریخ ثبت لینک ، تاریخ آخرین ارجاع یا بازدید از لینک ، تعداد بازدید

فایل redirect.php
این تابع مشخص می کند که کاربر با لینک کوتاهی که وارد کرده است می خواهد به لینک اصلی وارد شود یا آمار لینک را مشاهده کند .

فایل api.php
از اسمی که دارد برای endpoint استفاده می شود که درخواست های ajax به این فایل ختم می شود .
